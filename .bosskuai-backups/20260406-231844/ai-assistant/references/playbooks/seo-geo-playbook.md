# SEO and GEO Playbook

> If the request is general, ambiguous, or touches many files — ask clarifying yes/no questions **before acting**. Use numbered bullets with explicit answer format: e.g. `1-yes/no  2-A/B`.

Use this when improving discoverability through search engines and generative engines.

## Workflow

1. Define the search intent, question, or problem statement.
2. Map core entities, related topics, and supporting evidence.
3. Structure content so the direct answer appears early and clearly.
4. Improve answerability, citation-worthiness, and trust signals.
5. Recommend content clusters, landing pages, and update loops.

## Output expectation

- intent map
- target topics and entities
- content structure guidance
- SEO and GEO improvements
- measurement or iteration ideas
