# Financial Modeling Checklist

## Pre-work
- [ ] Model purpose stated: fundraising / internal planning / board reporting
- [ ] Time horizon defined: 12 / 24 / 36 months
- [ ] Key inputs collected: MRR, churn rate, ACV, headcount, burn rate, cash balance

## Revenue model
- [ ] Revenue streams all accounted for
- [ ] Growth assumptions stated explicitly (not just %)
- [ ] Churn / expansion revenue modeled separately
- [ ] Pricing tiers reflected accurately

## Cost model
- [ ] COGS separated from OpEx
- [ ] Headcount plan includes all loaded costs (salary + benefits + taxes)
- [ ] Infrastructure costs scale with revenue (not fixed)
- [ ] One-time vs recurring costs distinguished

## Unit economics
- [ ] CAC calculated (total S&M spend ÷ new customers)
- [ ] LTV calculated (ACV × gross margin ÷ churn rate)
- [ ] LTV:CAC ratio ≥ 3x target validated
- [ ] Payback period calculated

## Scenarios
- [ ] Base / optimistic / pessimistic cases defined
- [ ] Key assumptions varied per scenario (not just revenue ±20%)
- [ ] Runway calculated for each scenario
- [ ] Break-even month identified

## Quality gate
- [ ] Sanity checks: revenue ÷ headcount within industry range
- [ ] No circular references in spreadsheet
- [ ] Model reviewed by second person
- [ ] Key sensitivities documented (which assumptions most affect runway)
