# Documentation Lookup Checklist

## Pre-work
- [ ] Library / framework name confirmed (exact package name)
- [ ] Version pinned or latest specified
- [ ] Specific question scoped: function name / config option / error message

## Lookup
- [ ] Context7 MCP used: resolve-library-id → get-library-docs
- [ ] Relevant section extracted (not entire dump)
- [ ] Code example confirmed against the target version
- [ ] Official docs URL cited

## Quality gate
- [ ] Answer validated against actual version in project (check package.json / go.mod / requirements.txt)
- [ ] Deprecated APIs flagged if applicable
- [ ] Migration guide noted if user is on old version
