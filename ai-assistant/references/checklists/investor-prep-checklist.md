# Investor Prep Checklist

## Pre-work
- [ ] Audience identified: VC / angel / accelerator / strategic
- [ ] Stage context: pre-seed / seed / Series A
- [ ] Core narrative drafted (problem → solution → traction → ask)
- [ ] Financial model current (last updated < 30 days)
- [ ] Comparable exits / comps identified

## Pitch deck
- [ ] Slide order: problem → solution → market → product → traction → team → financials → ask
- [ ] Each slide passes the "so what?" test
- [ ] No slide > 6 bullet points
- [ ] Traction slide has real numbers (not % growth without base)
- [ ] Team slide shows relevant domain expertise
- [ ] Ask is specific: $ amount + use of funds + milestones unlocked

## Investor memo
- [ ] One-pager version exists for cold outreach
- [ ] Full memo: executive summary, problem, solution, market, business model, traction, team, financials, ask
- [ ] Key risks named and mitigated (investors will find them anyway)
- [ ] No unsubstantiated market size claims

## Due diligence readiness
- [ ] Cap table is clean and up to date
- [ ] Incorporation docs accessible
- [ ] IP ownership confirmed (no contractor IP issues)
- [ ] Key contracts and customer agreements organized
- [ ] Financial statements (P&L, balance sheet, cash flow) for past 12 months

## Quality gate
- [ ] Pitch rehearsed: opening hook < 30 seconds
- [ ] Top 10 likely questions prepared with honest answers
- [ ] No vanity metrics without context
- [ ] Follow-up materials ready (data room link, one-pager, deck)
