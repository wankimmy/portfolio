# Deep Research Checklist

## Pre-work
- [ ] Research goal stated in one clear sentence
- [ ] Decision this research informs is identified
- [ ] Scope confirmed: technology review / market sizing / company profile / regulatory landscape / other
- [ ] 3–5 sub-questions decomposed from the main goal
- [ ] Exa MCP confirmed available (or fallback noted)
- [ ] Firecrawl available for deep-reads (or snippet-only mode noted)

## During
- [ ] At least two independent searches per sub-question (varied query phrasing)
- [ ] Primary sources identified for each sub-question (official docs, filings, product pages)
- [ ] Secondary sources identified (community, reviews, journalism)
- [ ] 3–5 key sources deep-read via Firecrawl / WebFetch (not just snippets)
- [ ] Every URL and retrieval date logged in the sources list
- [ ] Material claims cross-referenced across at least two independent sources
- [ ] Contradictions between sources flagged explicitly
- [ ] Confidence level assigned to each finding: HIGH / MEDIUM / LOW
- [ ] Gaps and unanswerable questions documented

## Quality gate
- [ ] Every factual assertion has an inline citation
- [ ] No HIGH-confidence claims without two independent sources
- [ ] Recency checked — sources within last 12 months for time-sensitive claims
- [ ] Executive summary answers the original research goal
- [ ] Gaps section is honest and complete — nothing papered over
- [ ] Methodology note includes queries used, tools used, date range
- [ ] Output reviewed for conflation of snippets vs. full-source reads
